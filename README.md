# DevisFlash

Le devis bâtiment, dicté en 30 secondes.

Application web pour artisans du BTP : dictée vocale du chantier, génération du
devis, comparaison des prix matériaux chez les enseignes, export du devis.

**Site en ligne :** https://melvin84700.github.io/devisflash-app/

---

Ce dépôt contient uniquement l'application publiée (`index.html`).
Le code de travail, les tests et le bridge IA sont gérés à part.
